Way2API Tutorial

In order to get a brief idea of what the Way2API is, please read the introduction on this software. To get started with, you need a Java project to use Way2API. Follow the steps given below to send a text message using this API:

  1. Add the jar file(way2api.jar) to the classpath of the project or as a library using any IDE such as Netbeans and Eclipse.
  2. Withing the source file where you want to use Way2API, you need to import two classes - Account and Dispatcher. These classes can be imported using the statements
    import way2api.model.Account;
    import way2api.Dispatcher;
  3. In your source code, you must first create an account corresponding to the Way2SMS account, you wish to use for sending text messages. To do so, create an object of the Account class with the username and password as the first and second parameters. You can also create an empty object and set the account with the methods setUsername() and setPassword().
    Account account=new Account("username","password");
    or,
    Account account=new Account();
    account.setUsername("username");
    account.setPassword("password");
  4. You then need to create an object of the Dispatcher class, optionally specifying the account object as a parameter. You may add the account object later on using the method setAccount().
    Dispatcher dispatcher=new Dispatcher(account);
    or,
    Dispatcher dispatcher=new Dispatcher();
    dispatcher.setAccount(account);
  5. To send a SMS, you may choose one of the two procedures: login-sendmessage-logout or sendquickmessage. To send multiple texts, it is recommended to use the former procedure. To send a quick message, you need to call the Dispatcher method sendQuickMessage() with the parameters, mobile_number and text.
    dispatcher.sendQuickMessage("9876543210","this is a message");
  6. To send multiple messages, you need to call the method login() on the dispatcher, and then send the messages by invoking the method sendMessage() for each text message, eventually followed by calling logout() on the dispatcher.
    dispatcher.connect();
    dispatcher.login();
    dispatcher.sendMessage("9876543210","this is a message");
    dispatcher.logout();
