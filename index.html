Way2API

Way2API is an API for sending free text messages(SMS) via the internet using the services of the popular website www.way2sms.com. This API is built for the Java programming language, and can be integrated into any Java project. Please refer to the tutorial on using this API. Way2API is not currently licensed and is basically a hack for the standard texting procedure via the service provider's website. The sustainability of this software is doubtful as the API solely depends on the web application hosted by Way2SMS. If any changes are introduced into the web application, this API may cease to work. Please notify me if that occurs.
